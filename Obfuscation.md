I.	Understanding Obfuscation
1.	Definition
-	Obfuscation is the act of making the code of a program hard to discover or understand by both human and computers but still remained how the program works
2.	Why to choose?
-	Applying obfuscation can helps owner of the program hide it’s **readable** functions to everyone, especially developer who wanted to steal their codes.
-	Obfuscations can help our program more lighter because the LOC (Line of Code) chopped down so much, improved speed of loading.
-	Costs more time to understanding code for unauthorized person.
3.	Is Obfuscation is a type of encryption?
-	TL;DR: No because it just only make the codes likely unfriendly with human and make them confuse, still able to de-Obfuscated using various tools.
II.	So…. Is bad boys apply Obfuscation in their malware and how does that effected?
-	Short answer: Yes, from my research that they also using that technique in their source code just for make sure that the analyst cost more time to understanding how this code is works.
